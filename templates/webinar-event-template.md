# Webinar Event Page/Post — Template

A reusable template for announcing an **X-CITED Digital Community of Practice** webinar event.
Use this for different speakers, dates, times, and topics by replacing `{{placeholders}}`.

---

## How to use

1. Replace all `{{placeholders}}`.
2. Keep or remove each `[OPTIONAL: …]` section as needed.
3. Keep headings to maintain a consistent event format.
4. Share as a webpage post, event page, or email announcement.

### Placeholders

| Placeholder | What it is | Example |
|---|---|---|
| `{{TOPIC_TITLE}}` | Webinar/session title | Effectively Communicating with Data |
| `{{SPEAKER_NAME}}` | Speaker full name | Dan Isaac |
| `{{SHORT_INTRO}}` | 1–2 sentence summary of what attendees will learn | Explore why audience and narrative shape effective data communication |
| `{{DATE}}` | Event date | Thursday 14 November 2026 |
| `{{TIME}}` | Event time and timezone | 13:00–14:00 (UK time) |
